## Getting Started

- Navigate to the project folder
- Install dependencies

```bash
npm i
```

Start the development server:

```bash
npm run dev
# lub
yarn dev
# lub
pnpm dev
# lub
bun dev
```

Open http://localhost:3000 in your browser to see the page.
